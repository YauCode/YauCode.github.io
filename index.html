CLONEFLIX - Netflix Clone Project

CLONEFLIX is a FullStack Project of Netflix Clone. This Project was created using Typescript, React, NextJS, TailwindCSS and Prisma ORM.

Project Features:

  • Created using NextJS framework

  • Type-Safe Code: Typescript

  • Modern styling, effects and animations using TailwindCSS

  • Fully responsive design on all screens

  • Registration with email, username, password

  • Authentication with NextAuth: Google and Github Login

  • Cookie based Authentication

  • Database creating and updating: MongoDB via Prisma ORM

  • React SWR for Data fetching

  • Zustand state management

  • Favorite Movies List

Technologies, frameworks and libraries used to create this project: NextJS, TypeScript, React, TailwindCSS, NextAuth, MongoDB, Prisma ORM, React SWR, Zustand

Visit GITHUB Page

Music Web Service

MUSIC WEB SERVICE is a Full-Stack Project of Spotify Clone. This Project was created using Next.js 13.4 App Router: React, Typescript, TailwindCSS, Supabase, PostgreSQL, Stripe.
PROJECT DEMO on VERCEL

For DEMO use Stripe Testing Cards

Project Features:

  • Created using NextJS 13.4 framework

  • Type-Safe Code: Typescript

  • Modern styling, effects and animations using TailwindCSS

  • Fully responsive design on all screens

  • Registration with email, username, password

  • Credential Authentication with Supabase: registration via email, registration confirmation link on email, forgot password functionality, magic link functionality

  • Github authentication integration

  • Songs and images upload using Supabase storage with PostgreSQL

  • Play songs audio

  • Playlists / Liked songs system

  • Advanced Player component

  • Client form validation and handling using react-hook-form

  • Stripe integration: recurring payments and subscriptions cancelling

  • Server error handling with react-toast
